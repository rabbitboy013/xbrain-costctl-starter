Cost command implemented.

Manual command to capture real Cost Explorer output:

python costctl.py --region us-west-2 cost --tag Application=CostCtlTest --days 7

Note:
Cost Explorer data can lag 8-24 hours and requires activated cost allocation tags.
In this submission, the real AWS evidence captured is focused on read-only list output
for EC2, EBS volume, and S3 so existing shared services are not modified.
